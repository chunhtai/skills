# implement_feature

Implement a new feature based on a description or specification.

## Parameters

- **description** (string, required): Description of the feature to implement.
- **file_path** (string, optional): Path to the file or directory where the implementation should go.
- **related_files** (array, optional): List of related file paths for context.

## Instructions

Implement the described feature following these steps:
1. Understand the requirements and clarify any ambiguities
2. Identify the affected files and components
3. Design the solution before writing code
4. Implement the feature following existing code patterns and conventions
5. Write or update unit tests to validate the implementation
6. Update relevant documentation
7. Ensure no existing functionality is broken

Make the smallest change that satisfies the requirements.
